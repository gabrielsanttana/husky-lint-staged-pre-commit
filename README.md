## 🚀 Installation

```bash
$ git clone https://github.com/gabrielsanttana/husky-lint-staged-pre-commit
$ cd husky-lint-staged-pre-commit
$ yarn
```

## 🧪 Commits

```bash
$ touch src/test.ts
$ git add test.ts
$ git commit -m "Checking pre-commit workflow"
```
